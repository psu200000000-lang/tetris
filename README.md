# 테트리스 (TETRIS)

고전 테트리스 게임을 Python과 Pygame으로 구현한 프로젝트입니다.

[![Python 3.7+](https://img.shields.io/badge/python-3.7+-blue.svg)](https://www.python.org/downloads/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Status](https://img.shields.io/badge/status-active-brightgreen.svg)]()

## 📸 스크린샷

게임은 클래식 테트리스 스타일의 3D 입체 블록으로 구성되어 있습니다.

## ✨ 주요 기능

- **클래식 테트리스 게임플레이**: 7가지 다양한 테트로미노 피스
- **칼라풀한 블록**: 각 피스마다 고유한 색상
  - I (시안), O (노랑), T (자주색), L (파랑), J (주황), S (초록), Z (빨강)
- **점수 시스템**: 라인 제거 시 점수 획득
- **난이도 조절**: 5 라인마다 레벨 업, 속도 증가
- **사운드 효과**: 라인 제거 시 벽돌 소리
- **게임 오버 애니메이션**: 버튼이 커졌다 작아졌다 반복
- **3D 입체감**: 블록에 밝은 선과 어두운 선으로 깊이감 표현
- **일시정지 기능**: 스페이스바로 게임 일시정지
- **빠른 낙하**: 아래 화살표 꾹 누르면 빠르게 내려감

### 📦 필수 사항

## 📊 게임 정보 화면

오른쪽 패널에서 다음 정보를 확인할 수 있습니다:

- **SCORE**: 현재 점수 (6자리)
- **LEVEL**: 현재 레벨 (5 라인마다 1씩 증가)
- **LINES**: 제거한 라인 수
- **NEXT**: 다음에 나올 피스 미리보기
- **PAUSED**: 일시정지 상태 표시

## 🎯 게임 규칙

1. 떨어지는 블록을 조작하여 가로 라인을 완성합니다
2. 완성된 라인은 제거되고 위의 블록들이 내려옵니다
3. 새 블록이 맨 위에서 기존 블록과 충돌하면 게임 오버입니다
4. 5 라인을 제거할 때마다 레벨이 올라가고 속도가 증가합니다

## 🎵 사운드

- 라인 제거 시 **벽돌 부서지는 소리** 재생

## 📝 코드 구조

```
tetris.py
├── create_block_sound()          # 사운드 생성
├── Button 클래스                  # 게임 버튼
└── TetrisGame 클래스             # 메인 게임 클래스
    ├── create_new_piece()        # 새 피스 생성
    ├── move_piece()              # 피스 이동
    ├── rotate_piece()            # 피스 회전
    ├── lock_piece()              # 피스 고정
    ├── clear_lines()             # 라인 제거
    ├── draw_*()                  # 렌더링 함수들
    └── run()                     # 메인 루프
```

## 🔄 게임 루프

1. **이벤트 처리** (handle_events): 키보드/마우스 입력
2. **게임 업데이트** (update): 블록 낙하, 충돌 감지
3. **렌더링** (draw): 화면 그리기
4. **프레임 제어** (clock.tick(60)): 60 FPS 유지
